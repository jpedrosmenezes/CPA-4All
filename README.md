# 🎮 CPA 4All — Controle de Performance Acessível

> Projeto 1 · Grupo 16 · CESAR School  
> Tornando o universo gamer inclusivo para pessoas com Distrofia Muscular.

---

## 📖 Sobre o Projeto

A **Distrofia Muscular** é um grupo de doenças genéticas caracterizadas pela degeneração progressiva dos músculos do corpo. Pessoas com essa condição frequentemente enfrentam barreiras para jogar videogames, pois os controles tradicionais exigem coordenação motora fina e força manual que muitas vezes está comprometida — além do alto custo de tecnologias assistivas já existentes no mercado.

O **CPA 4All** (Controle de Performance Acessível para Todos) é um controle alternativo baseado em **Arduino**, projetado para exigir o mínimo de esforço físico e ter custo acessível. O dispositivo conta com botões de arcade sensíveis ao toque, layout plano e ergonomia pensada para garantir **autonomia, conexão e o direito ao lazer** para todos os jogadores, independentemente de suas limitações físicas.

---

## 🚧 O Desafio

Pessoas com distrofia muscular enfrentam uma dupla barreira ao tentar jogar videogames:

- **Barreira motora** — controles convencionais exigem força manual e coordenação fina que essa condição progressivamente compromete.
- **Barreira financeira** — soluções assistivas como o Microsoft Adaptive Controller têm custo elevado, tornando o acesso ao entretenimento interativo ainda mais restrito.

A ausência de alternativas acessíveis e acessíveis financeiramente contribui para o isolamento social desse público, impactando diretamente o bem-estar psicológico.

- **Metodologias:** Design Thinking, Método 635, Matriz de Priorização, Benchmarking, Canvas do Problema Cabeludo

---

## 💡 Diferenciais do CPA 4All

- ✅ **Baixo esforço físico** — botões sensíveis que não exigem força manual
- ✅ **Layout plano** — ergonomia projetada para uso com mobilidade reduzida
- ✅ **Custo acessível** — baseado em componentes de baixo custo (Arduino + arcade)
- ✅ **Foco em nicho real** — desenvolvido com o contexto da distrofia muscular em mente
